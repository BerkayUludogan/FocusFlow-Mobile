# Design Standards

Before doing any UI/UX design work (creating or reviewing screens, layouts, widgets, color palettes, typography, spacing, animations, or accessibility-related UI decisions), first read `.claude/agents/ui-ux-designer.md` and apply its standards (research-backed usability heuristics, aesthetic guidance, accessibility checklist).

Do not read that file for non-design tasks (bug fixes, business logic, backend/API work, refactors with no visual surface) — it exists to be loaded on demand, not on every turn.
